Download / Install / Run Ollama

01 - Download / Install

Go to https://ollama.com/ copy paste command in terminal

curl -fsSL https://ollama.com/install.sh | sh

02 - Run

Run the command

ollama

If it doesn't work, open the Ollama app in applications or spotlight search

03 -

Go to ollama.com/search to find a model. Specifics how to use in 2. Models (press next button) or 'E' on keyboard jump to 'next lesson(E)' 'P' on keyboard jump to 'previous lesson(p)' 'S' side bar so on
